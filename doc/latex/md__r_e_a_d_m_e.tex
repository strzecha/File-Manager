Program do porządkowania plików w systemie Linux. Ma on za zadanie przeszukać główny katalog X razem z jego podkatalogami, oraz inne katalogi Y1, Y2, ... wraz z ich podkatalogami (o podobnej strukturze), a następnie uporządkować te katalogi w następujący sposób\+:
\begin{DoxyItemize}
\item w katalogu X znalazły się wszystkie pliki,
\item zlikwidować duplikaty czyli sposród plików o identycznej zawartości zasugerować pozostawienie tylko najstarszego (nowsze daty są zapewne datami utworzenia kopii),
\item zaproponować skasowanie plików pustych i tymczasowych,
\item w przypadku plików o tej samej nazwie zasugerować pozostawienie nowszego,
\item zasugerować ujednolicenie atrybutów np. rw-\/r–r–,
\item zasugerować przemianowanie plików, których nazwy zawierają znaki mogące komplikować operowanie nimi (np. ’\+:’, ’”’, ’.\+’, ’;’, ’$\ast$’, ’?’, ’\$’, ’\+::’, ’‘’, ’$\vert$’, ’\textbackslash{}’, ...) zastępując te znaki zdefiniowanym substytutem np. \+\_\+
\end{DoxyItemize}

Program potrafi wyszukiwać\+:
\begin{DoxyItemize}
\item pliki o identycznej zawartości (choć niekoniecznie identycznej nazwie i niekonieczne w analogicznej pozycji w drzewie),
\item pliki puste,
\item pliki tymczasowe ($\ast$˜, $\ast$.tmp oraz inne rozszerzenia zdefiniowane przez użytkownika),
\item pliki o atrybutach innych niż zdefiniowane przez użytkownika,
\item pliki o nazwach zawierających kłopotliwe znaki.
\end{DoxyItemize}

Dla każdego znalezionego pliku program proponuje odpowiednią akcję\+:
\begin{DoxyItemize}
\item skasowanie duplikatu, pliku pustego lub tymczasowego,
\item zastąpienie starszej wersji nowszą (w przypadku plików z identyczną zawartością),
\item zastapienie wersji nowszej starszą (w przypadku plików z identyczną zawartością),
\item zmianę atrybutów,
\item zmianę nazwy,
\item pozostawienie bez zmian
\end{DoxyItemize}\hypertarget{md__r_e_a_d_m_e_autotoc_md1}{}\doxysection{Uruchomienie}\label{md__r_e_a_d_m_e_autotoc_md1}
